# Node.js Express App

This is a simple Node.js application using Express with a basic routing setup.

## Project Structure

```
.
├── app.js
├── package.json
├── README.md
└── routes
    └── index.js
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- Express server setup
- Modular routing
